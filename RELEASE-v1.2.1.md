## Toolbox v1.2.1

### ✨ 新增
- **检查更新**：设置页新增"检查更新"按钮，自动检测新版本并下载安装
- **双源下载**：优先从 Gitee 下载，失败自动切换 GitHub，提升更新成功率
- **应用图标**：更换为新图标

### 🎨 改进
- 工具页面增加分类标签栏（全部 / 效率 / 财务 / 网络 / 生活 / 健康 / 图片）
- 首页卡片布局优化，图标与文字垂直居中
- 拼豆图纸网格自适应屏幕宽度，大尺寸不再被截断

### 🔧 修复
- IP 属地查询 HTTP 403 问题（更换 API 为 api.ip.sb）
- 拼豆图纸大于 16×16 时显示不全的问题
- 底部导航点击跳转异常
