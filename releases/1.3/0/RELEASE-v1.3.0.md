# Toolbox v1.3.0 — 密码箱 & 设置页 UI 全面重设计

### ✨ 视觉统一

将密码箱页和设置页的视觉风格全面对齐首页与工具页，形成统一的「浅色简约卡片风」。

**密码箱页**
- 新增页面顶部居中标题「密码箱」与副标题说明
- 解锁卡片新增靛蓝色圆形锁图标、生物识别（指纹）入口按钮
- 解锁失败时输入框红色边框 + 抖动动画反馈
- 设置主密码卡片同样增加锁图标，排版居中优化

**设置页**

- 新增页面标题
- 外观选项：从 `FilterChip` 改为「卡片式单选」— 每个选项含 48dp 彩色圆形图标、选项文字、单选圆圈指示器
  - 跟随系统（灰色手机图标）
  - 浅色（琥珀色太阳图标）
  - 深色（深蓝色月亮图标）
- 数据管理：从纯文字按钮改为带图标列表卡片，含绿色/蓝色圆形图标、功能描述、右箭头
- 版本号改为白底圆角卡片展示，信息结构更清晰

**风格统一**
- 所有卡片圆角统一为 16dp
- 图标风格统一为 48dp 彩色圆形背景 + 白色图标，与首页工具网格一致
- 按钮统一使用深灰色 `#333333` 圆角矩形
- 页面背景统一为浅灰 `#F5F5F7`
- 未引入任何新设计语言（无渐变、毛玻璃、深色变体）

### 🛠 技术栈

- **语言**: Kotlin
- **UI**: Jetpack Compose + Material3
- **架构**: MVVM + Hilt DI
- **数据库**: Room
- **最小改动原则**: 仅修改 2 个 UI 文件（`PasswordScreen.kt`、`SettingsScreen.kt`），未改通用组件、主题、导航
