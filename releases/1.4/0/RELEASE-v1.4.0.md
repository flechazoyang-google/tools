# Toolbox v1.4.0 — 5个新工具 & 交互大升级

### ✨ 新增
- **二维码生成** — 文本/链接转二维码，支持保存到相册和分享，尺寸可调
- **亲戚称呼计算器** — 计算器风格按钮交互，两步链式选择，自动计算正确称呼
- **经期记录** — 日历标记+自动预测下次经期，支持 DataStore 持久化
- **去水印** — 选区定位水印区域，周边像素采样填充去除
- **加水印** — 自定义文字水印，可调位置/大小/透明度/旋转

### 🛠 功能增强
- **工具收藏系统** — 工具页星标收藏，首页「收藏工具」展示最多3个，支持独立收藏页
- **密码箱会话超时** — 解锁后5分钟内切换Tab无需重输密码
- **密码箱+倒数日 编辑功能** — 支持修改已保存的密码和事件
- **删除确认弹窗** — 倒数日和密码箱删除操作前弹出确认对话框
- **拼豆图纸重构** — 工程图纸风格：方格网格+行列坐标+颜色代码（A1~O2）+ 代码/色块双模式
- **密码箱解锁错误动画** — 密码错误时输入框抖动反馈

### 🛠 修复
- 修复二维码生成因未捕获 WriterException 导致的崩溃
- 修复 LazyVerticalGrid 嵌套 verticalScroll 导致的 Compose 崩溃
- 修复 FavoritesScreen 卡片点击无导航跳转
- 修复 FavoritesScreen 缺少返回按钮
- 修复拼豆图纸大图转换在主线程执行导致的 ANR 风险
- 修复二维码 Bitmap 使用 RGB_565 格式画质差的问题
- 修复 FeaturedToolCard 参数顺序导致 trailing lambda 错配

### 🎨 UI/UX 优化
- 首页「常用工具」限制为6个（3行×2列）
- 工具页增加搜索栏、分类计数徽章、卡片入场动画
- 亲戚称呼计算器改为计算器风格按钮网格（替代下拉菜单）
- 空状态增加图标（倒数日/收藏/密码箱）
- 二维码生成改为 ARGB_8888 格式，画质提升
