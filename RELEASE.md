# Toolbox v1.4.3 — 首页收藏快捷入口

一款简约实用的 Android 工具箱应用，集成 17 种日常工具，基于 Kotlin + Jetpack Compose 构建。

## ✨ 功能

| 工具 | 说明 |
|------|------|
| 🧮 计算器 | 四则运算与括号计算 |
| 💱 汇率换算 | 本地汇率实时换算 |
| 🌐 IP 属地查询 | 查询 IP 地址归属地信息 |
| 📅 倒数日 · 纪念日 | 支持倒数日、纪念日、生日三种类型 |
| 🔐 密码箱 | 本地加密存储密码，支持指纹解锁 / 主密码 |
| 🏋️ BMI 计算器 | 身体质量指数计算 |
| 🖼️ 九宫格切图 | 图片切成 3×3 九宫格并保存到相册 |
| ⏱ 番茄钟 | 25+5 专注计时，支持短休/长休 |
| 🧩 拼豆图纸 | 图片转拼豆像素图纸，支持工程图纸风格 |
| 🔤 Base64 编解码 | 文本 Base64 编码/解码、一键复制 |
| ⏰ 时间戳转换 | Unix 时间戳与日期互转 |
| 📱 二维码生成 | 文本/链接转二维码，保存与分享 |
| 👪 亲戚称呼计算 | 计算器风格，链式选择计算正确称呼 |
| 🩸 经期记录 | 日历标记经期，自动预测下次周期 |
| 🧹 去水印 | 选区定位水印，周边像素填充去除 |
| 💧 加水印 | 自定义文字水印，位置/大小/透明度/旋转可调 |
| ⭐ 收藏 | 跨页面收藏工具，独立收藏列表页 |

## 📋 v1.4.3 更新内容

- ✨ 首页工具标签栏新增「我的收藏」快捷入口芯片，带星标图标和右箭头，点击直达收藏页

- 🐛 修复密钥库初始化异常未捕获
- ✨ 列表增删重排动画、按钮按压反馈、导航图标渐变
- ✨ 搜索清除按钮动画
- 🎨 首页收藏入口始终可见

## 🎨 设计

- 黑白灰中性色系，低饱和语义色区分功能
- 卡片使用 1px 边框替代阴影，按压缩放反馈
- 底部导航选中态明确，未选中弱化
- 支持浅色/深色模式
- 工程图纸风格拼豆网格

## 🛠 技术栈

- **语言**: Kotlin
- **UI**: Jetpack Compose + Material3
- **架构**: MVVM + Hilt DI
- **数据库**: Room
- **网络**: Retrofit + OkHttp
- **导航**: Navigation Compose
- **最低 SDK**: Android 7.0 (API 24)
