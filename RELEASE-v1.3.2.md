# Toolbox v1.3.2 — 搜索功能 & 分类切换

### ✨ 新增
- **工具搜索**：工具页搜索栏点击展开为输入框，支持按工具名称和描述实时模糊过滤
- **左右滑动切换分类**：工具页网格区域左右滑动即可切换「全部 / 效率 / 财务 / 网络 / 生活 / 健康 / 图片」分类
- **搜索键盘快捷键**：输入文字后键盘回车即可收起键盘

### 🛠 修复
- **分类标签状态重置**：修复 LazyVerticalGrid 嵌套 verticalScroll 导致点击其他分类后自动跳回「全部」的 bug
- 分类标签栏改用非惰性网格布局，消除 Compose 嵌套滚动冲突
